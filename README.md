# C172 World Hop

A cinematic browser flight simulator built with Vite, TypeScript, and Three.js.

## Run

```bash
npm install
npm run dev
```

## Controls

- `W` / `S` or arrow up/down: pitch
- `A` / `D` or arrow left/right: roll
- `Q` / `E`: rudder
- `+` / `-`: throttle
- `F`: add flaps
- `Shift+F`: retract flaps
- `Space`: brake on runway
- `P`: autopilot
- `C`: camera mode
- `Esc`: pause and open the flight deck
- `R`: reset current leg
